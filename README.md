# Cyber Incident Management System (Java Project)

## Overview
This is a Java based Cyber Security Incident Management System developed as a university project.  
The system allows users to log, manage, and store cyber security incidents.

It supports:
- Console based interface
- GUI based interface
- File handling for persistent data storage

## Technologies Used
- Java
- OOP (Object-Oriented Programming)
- File I/O
- Java Swing (GUI)

## Features
- Add new incident
- View incident list
- Search incidents
- Store incidents in file
- User friendly text & GUI interface

## What I Learned
- OOP design principles
- File handling in Java
- GUI development
- Structuring multi-class Java applications

## How to Run
1. Compile: `javac Main.java`
2. Run: `java Main`
